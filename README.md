# Hangman Game 🎮

## Project Description
This is a simple text-based Hangman Game developed using Python. The player selects a category and guesses the hidden word one letter at a time. The game ends when the player guesses the word correctly or uses all 6 chances.

## Features
- 3 Categories (Fruits, Animals, Programming)
- Random word selection
- 6 incorrect attempts
- Easy console-based gameplay
- Win and Game Over messages

## Technologies Used
- Python 3
- Random Module

## How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Open the project in VS Code.
4. Run the file:
   ```
   python hangman.py
   ```
